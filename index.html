Backing Tracker Application

A few years ago, the local musician, who I help run a Wordpress music lesson site, wanted to make a web app that allows a subscribed user to build their own backing tracks that they can listen and play along to. Initial design was done by me and a colleage, but after he left I built the rest myself. We learned and ended up using an open-source Javascript platform, MeteorJs (or just 'Meteor'), to create a responsive Javascript web application.

Tech Info: User account data is sent over from Wordpress's current login info, and user song data is stored in a Mongo database. Meteor was able to help client-server integration, such as implementing RESTful API and Database for the data storage and transfers. In order to deploy it onto the domain we made for it, it was bundled into a Node.js application, and ran it on a Phusion Passenger installation.

Computer Game: D.I.D

D.I.D (short for Dissociative Identity Disorder) is a game I helped to create in a summer internship. I was the lead programmer, and I worked alongside four other people I met at the start of the internship, each with different areas of talent. In the span of less than 2 weeks, we were introduced to the Unity game engine, worked with Unity veterans who taught us, had mock conferences and meetings, and worked full-time in an office setting to eventually produce this game.

What made this experience special? Some guests of various educational sociological backgrounds were called in at the beginning, and part of the challenge of this internship was that each guest presented a social issue idea that they'd like to see implemented in a game, and each individual team had to find a way to incorporate the one assigned to them. In our case, the social concept was 'implicit bias'. Thus, the puzzle aspect of this game involved piecing different personalities together based on journal entries collected, without making assumptions. Additionally, each personality was intended to allude to the concept of dissassociative identity disorder, with the player potentially implicitly attributing each to one person as one might suspect D.I.D. to do. This in turn drew attention to the assumptions we may make about the disorder itself. Also, at the end of the 2 weeks, students from each internship program got to show their final products to all the other students, meaning that this game was actually the first video game experience for some people!
